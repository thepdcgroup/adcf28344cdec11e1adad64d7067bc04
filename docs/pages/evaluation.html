Evaluation
GitHub is a ____ hosting service for software development and version control?


web-based
A central storage for code projects on GitHub is called a ____?


repository
GitHub uses the distributed version control system called ____?


Git
The action of saving changes to a repository in Git is known as a ____?


commit
A ____ in GitHub allows you to experiment with changes without affecting the main codebase?


branch
A ____ is a request to merge your branch changes into the main branch?


pull request
GitHub provides a feature to track bugs and feature requests, known as ____?


issues
The process of copying a repository from GitHub to your local machine is called ____?


clone
A ____ is used to make a personal copy of someone else's repository on GitHub?


fork
Merging changes from different branches is performed using a ____?


merge
To update your local repository with changes from the remote repository, you use ____?


pull
After making local changes, you can send your updates to GitHub using ____?


push
The graphical interface for code collaboration provided by GitHub is known as ____?


GitHub Desktop
The command line tool to interact with Git repositories is called the ____?


Git CLI
A ____ is used to undo changes by creating a new commit that reverts previous commits?


revert
GitHub supports continuous integration with a feature called ____?


GitHub Actions
The process of incorporating changes from one branch into another is often referred to as ____?


merging
When you open a pull request, the process of reviewing and discussing the changes is known as ____?


code review
GitHub provides a way to manage project tasks with a feature called ____?


Projects
A secure token used for authentication on GitHub is known as a ____?


Personal Access Token
To resolve conflicts in a repository, you might use a ____?


merge tool
A file that outlines guidelines for contributing to a repository is called ____?


CONTRIBUTING.md
The file that often contains an introduction and instructions for a GitHub repository is called the ____?


README
GitHub offers a feature to host static websites directly from a repository, known as ____?


GitHub Pages
A written summary of project changes, typically accompanying a pull request, is called a ____?


changelog
